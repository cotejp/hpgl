Plotter Class

Defined in: src/hpgl.js:12
Module: hpgl

The Plotter class provides methods to interact with an HPGL-compatible plotter such as those made by HP starting in the 1980s. Various other makers also use or support the HPGL protocol.

By default, this library uses the Node.js serialport module for serial communication. This module offers native support on Mac, Linux and Windows. The library can also use the browser-serialport module which uses the chrome.serial API (only available in Chrome Apps and NW.js). For debugging purposes, it can also use the virtual-serialport.
