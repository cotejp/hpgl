Plotter

new Plotter()

Source:

The Plotter class provides methods to interact with an HPGL-compatible plotter such as those made by HP starting in the 1980s. Various other makers also use or support the HPGL protocol (Calcomp, for example).

Transport Layer

By default, this library uses the Node.js serialport module for serial communication. This module offers native support on Mac, Linux and Windows. The library can also use the browser-serialport module which uses the chrome.serial API (only available in Chrome Apps and NW.js). For debugging purposes, it can also use the virtual-serialport.

Coordinate Sytem

The coordinate system is anchored in the top-left of the paper sheet. Positive x goes right and positive y goes down. Some plotters work differently but I found it easier to stick with the computer screen standard.

setVelocity(velocityopt) → {Plotter}

Source:
  • hpgl.js, line 636

Sets the velocity of the plotting pen. When the velocity parameter is set to 1, the velocity will be at its maximum of 38.1cm/s (default). So, if you set the velocity parameter to 0.1, the actual velocity will be 3.81cm/s.

Any value equal or lower than 0 and any value above 1 will trigger the use of the default velocity.

Parameters:
Name Type Attributes Default Description
velocity number <optional>
 1.0

A decimal number between 0 and 1.
